# Applications Directory

This directory contains KGAS application files and entry points.

## Contents

### KGAS Application Files
- `kgas_*.py` - Various KGAS application implementations and integrations
- `streamlit_app.py` - Streamlit web application interface
- `main.py` - Main application entry point

### Integration Applications
- Files demonstrating KGAS integration with other systems
- Claude Code integration examples
- MCP server implementations

## Usage

Run applications from the repository root:
```bash
# Streamlit web interface
streamlit run applications/streamlit_app.py

# Main application
python applications/main.py

# KGAS MCP server
python applications/kgas_mcp_server.py
```
